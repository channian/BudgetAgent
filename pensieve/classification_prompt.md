# Notes
- 「監控」（SCADA / PLC / 中控 / CCTV 等）已併入【資安】，不再是獨立類別；凡監控/監測相關一律歸【資安】。

# Conflict Resolution (Priority)
1. 【Relayout】最高優先：凡涉及「家具配置、隔間、空間規劃/調整、裝修、拆除、機台進駐、Class增設、Relayout」，無視其他設備詞彙，一律判為【Relayout】。
2. 【二次配】次高優先：內容提及「二次配」關鍵字，優先歸類。
3. 【建管】特定定義：僅限涉及「法規、結構安全、違章」等關鍵字。
4. 通用組件：法蘭、蝶閥、管帽等若無明確主體，歸類為【Relayout】。
5. 如果出現非常多的系統，但是有包含到 Relayout，那就是歸類為【Relayout】。
6. 專案名稱若有二個系統，以後者系統派報，例：ZONE II 真空機排氣異味改善 → 【抽氣】。
7. 如果有幾個不同系統，那就依照系統的預算報價，以金額最高的系統為主。

# Output Constraints
- 嚴禁輸出任何解釋、標點符號、括號、前導詞（如「根據...」）或「無法判定」。
- 格式：僅輸出類別名稱本身（例如：水務）。
