四個階段說明

階段腳本輸入輸出
① Pipeline 1 pipeline_1.py 網路資料夾（各案件文件） 分類 + 全文，結果複製到剪貼簿
② 線上 AI 審核 手動 / RPA 剪貼簿（P1 輸出：分類 + 全文）→ 貼到 ChatGPT/Claude 每案件：判定系統/判定類別/負責專家（原樣帶回）+ 最終決策 + 信心分數 + 原因
③ 複製 AI 輸出 使用者 Ctrl+C 線上 AI 輸出 JSON 放入剪貼簿，供 P2 讀取
④ Pipeline 2 pipeline_2.py / pipeline_2_normalize.py 剪貼簿（AI 審核結果，含分類欄位） 直接寫入 PostgreSQL budget.budget_requests（不再讀取 system_defined.json）

DB 寫入去重邏輯

情境動作
DB 無此案件✅ INSERT，status = AI_REVIEW
DB 有，簽名相同（資料未變）⏭ 略過
DB 有，簽名不同，狀態 = AI_REVIEW / EXPERT_REVIEW / PENDING_ACTION🔄 UPDATE 更新 AI 欄位
DB 有，狀態 = CLOSED / REJECTED，重新送入📎 補送 INSERT（名稱加「（補送）」）
